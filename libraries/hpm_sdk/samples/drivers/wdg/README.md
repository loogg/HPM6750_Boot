# WDG API demo

## Overview

The demo provides the following features:
- WDG interrupt demo
- WDG timeout reset demo

## Board Setting

No special settings are required

## Running the example

When the example runs successfully, the board shows the following menu:

```shell
****************************************************************
*                                                              *
*  WDG Example Menu                                            *
*  1. Test WDG interrupt                                       *
*  2. Test WDG reset                                           *
*                                                              *
****************************************************************



```
