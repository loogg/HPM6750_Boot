# SPI_POLLING_MASTER
## 概述

spi_polling_master示例工程展示了SPI作为master使用polling方法进行板与板之间通信。
在这个示例工程中， 一个板子作为SPI master， 另一个板子作为SPI slave。 Master发送一定量数据到Slave并从Slave接收一定量的数据。

## 硬件设置

将两个板子的SPI引脚相连。
