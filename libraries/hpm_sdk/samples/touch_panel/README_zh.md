# Touch Panel
## 概述

该工程演示了触摸屏功能。

## 硬件设置

连接触摸屏到板上LCD接口

## 运行现象

当工程正确运行后，触摸屏幕，串口终端会输出如下信息：
```

> 0x14 failed to init GT911, try 0x5d
```
