# 等对称加解密算法AES
## 概述

**aes_example**示例工程展示了等对称加解密算法AES模式的实现。
- 此工程的等对称加解密算法采用AES128 NIST模式完成模块加解密测试。

## 硬件设置

- 无特殊设置
