# 等对称加解密算法AES-CTR
## 概述

**aes_ctr_prng_example**示例工程展示了等对称加解密算法AES-CTR模式的实现。
- 此工程的等对称加解密算法采用AES加密算法CTR-PRNG模式完成模块加解密测试。

## 硬件设置

- 无特殊设置

## 运行现象

当工程正确运行后，串口终端会输出如下信息：
```
> tc_start() - Performing CTR-PRNG tests:
> All CTR PRNG tests succeeded!
> ===================================================================
> PASS - main.
> ===================================================================
> PROJECT EXECUTION SUCCESSFUL
```
